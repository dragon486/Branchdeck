# Branchdeck for VS Code

Understand large codebases in hours, not weeks. 

Branchdeck brings AI Project Maps, Visual Call Flows, Impact Analysis, and Architecture Story Mode directly into your editor.

![Branchdeck Logo](media/icon.png)

## Features

- 🗺️ **AI Project Map**: Visualize folder structures grouped logically by domain features.
- ⛓️ **Visual Call Flow**: Interactively trace function call paths between endpoints, services, and queries.
- 🎯 **Impact Scope Analysis**: Predict refactoring risks and see caller metrics before changing code.
- 📖 **Architecture Story Mode**: Read natural language summaries of codebase features and data flows.

## Usage

1. Open any workspace in VS Code.
2. Open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
3. Run **`Branchdeck: Show AI Project Map`**.
